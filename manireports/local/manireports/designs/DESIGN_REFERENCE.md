# ManiReports Dashboard Design Reference

## Design Inspiration Sources

### Image 1: Clean Analytics Dashboard
- Minimalist KPI cards with trend indicators
- Large bar chart with dual metrics (Gross margin & Revenue)
- Donut chart for category breakdown
- Geographic distribution visualization
- Color scheme: Blue, Orange, Purple accents

### Image 2: Colorful Accounting Dashboard
- Vibrant colored KPI cards with icons (Orange, Blue, Red, Green)
- Time period selector (1D, 7D, 1M, 3M, All)
- Stacked bar chart (Cashflow visualization)
- Side-by-side metrics (Income vs Expense)
- Donut chart with legend (Project Status)
- Data table with status badges
- Color scheme: Orange, Blue, Red, Green, Purple

### Images 3-6: Additional Reference Patterns
- Sidebar navigation layouts
- Course progress indicators
- Activity feeds

- Upcoming deadlines widgets

## Design Principles Applied

1. **Visual Hierarchy**: KPIs first, then detailed analytics
2. **Color Coding**: Status indicators (Green=Good, Orange=Warning, Red=Alert)
3. **Responsive Layout**: Works on desktop and tablet
4. **Data Density**: Balance between information and whitespace
5. **User-Friendly**: Clear labels, intuitive navigation
6. **Accessibility**: High contrast, readable fonts

## Target Dashboards

- Admin Dashboard (system-wide analytics)
- Manager Dashboard (company/department metrics)
- Teacher Dashboard (course and student analytics)
- Student Dashboard (personal progress)
